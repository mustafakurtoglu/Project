  • BI Infrastructure
    February 14, 2023
    10 min read

    In the rapidly evolving landscape of data analysis, machine learning techniques stand at the forefront, offering powerful tools for extracting insights from vast pools of data. Yet, the application of these methods often stumbles upon a significant hurdle: the intensive manual effort required for data refinement and parameter optimization. This challenge is particularly pronounced in the realm of outlier detection—a critical process for identifying anomalies within data sets that, if overlooked, can lead to inaccurate analyses and misguided conclusions. My thesis, titled "Building Outlier Detection Framework by Using Automated Machine Learning Methods," addresses this crucial gap by proposing an advanced framework designed to streamline and automate the outlier detection process across various data types.
